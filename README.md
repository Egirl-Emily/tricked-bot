# Tricked-Bot 

A simple discord bot made for my [discord](https://discord.gg/mY8zTARu4g)

## Features

* rss feeds
* channel topic changer
* markdown renderer
* zipper
* qr code generator
* responders
* reddit post fetcher
* invite tracking

## Tech

* Bot is made in twilight
* database used is sqlite
